# PlayStation Store 游戏折扣 | 价格

支持浏览器扩展 支持 RSSBud

作者: @MisteryMonster

举例: https://rsshub.app/ps/product/UP9000-CUSA00552_00-THELASTOFUS00000

路由: /ps/:lang/product/:gridName

参数:

  • lang, 必选 -

    地区语言

  • gridName, 必选 -

    游戏的 grid 名

地区语言如 zh-hans-hk 代表香港区简体中文， zh-hant-tw 为台湾繁体中文。不同地区游戏 gridName 不同，非中文地区使用英文提示。

适用于 URL 如 https://store.playstation.com/zh-hans-hk/product/HP4497-CUSA16570_00-ASIAFULLGAME0000 (opens new window) 的游戏。

比如 PlayStation Store 香港简体中文区的 《赛博朋克 2077》 (opens new window) 的 lang 为 zh-hans-hk， gridName 为 HP4497-CUSA16570_00-ASIAFULLGAME0000
