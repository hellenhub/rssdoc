# CurseForge

# 文件更新

作者: @junfengP

举例: https://rsshub.app/curseforge/sc2/assets/taylor-mouses-stuff/files

路由: /curseforge/:gameid/:catalogid/:projectid/files

参数:

  • gameid, 必选 -

    游戏名，以https://www.curseforge.com/sc2/assets/taylor-mouses-stuff/files为例，sc2代表星际2

  • catalogid, 必选 -

    分类名，紧跟在游戏名后，如示例中assets

  • projectid, 必选 -

    项目名，紧跟在分类名后，如示例中taylor-mouses-stuff

例如：https://www.curseforge.com/sc2/assets/taylor-mouses-stuff/files 对应 /curseforge/sc2/assets/taylor-mouses-stuff/files

# MIUI

# MIUI 新版本发布

作者: @Indexyz

举例: https://rsshub.app/miui/aries

路由: /miui/:device/:type?/:region?

参数:

  • device, 必选 -

    设备的 codename 例如 小米 2s 为 aries. 国际版的 codename 一般以 _global 结尾. 可查阅 MIUI 线刷包下载页面

  • type, 可选 -

    类型

  • region, 可选 -

    地区, 默认为 cn

类型 type
稳定版 release
开发版 dev
地区 region
国内版 cn
国际版 global
