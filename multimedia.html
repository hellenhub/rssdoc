# 141JAV BT

支持 BT 支持浏览器扩展

作者: @cgkings

举例: https://rsshub.app/141jav/popular/30

路由: /141jav/:type/:key?

参数:

  • type, 必选 -

    类型

  • key, 可选 -

    关键词

类型

最新 热门 随机 指定演员 指定标签 指定日期
new popular random actress tag day

关键词

空 日期范围 演员名 标签名 日期
7 / 30 / 60 Yua%20Mikami Adult%20Awards YYYYMMDD

示例说明

  • /141jav/new

    仅当类型为 new popular 或 random 时关键词可为 空

  • /141jav/popular/30

    popular random 类型的关键词可填写 7 30 或 60 三个 日期范围 之一

  • /141jav/actress/Yua%20Mikami

    actress 类型的关键词必须填写 演员名 ，可在 此处 (opens new window) 演员单页链接中获取

  • /141jav/tag/Adult%20Awards

    tag 类型的关键词必须填写 标签名 且标签中的 / 必须替换为 %2F ，可在 此处 (opens new window) 标签单页链接中获取

  • /141jav/day/20200730

    day 类型的关键词必须填写 日期 ，按照示例写成形如 20200730 的格式

# Avgle

# 视频列表

作者: @I2IMk

举例: https://rsshub.app/avgle/videos

路由: /avgle/videos/:order?/:time?/:top?

参数:

  • order, 可选 -

    视频次序, bw 观看中 / mr 最新 / mv 最多观看 / tr 最高评分 / tf 最多收藏 / lg 最长, 默认 mv

  • time, 可选 -

    视频的添加时间, a 所有 / t 今天 / d 本周 / m 本月, 默认 m

  • top, 可选 -

    按次序获取的视频数, 不大于 250, 默认 30

# OneJAV BT

支持 BT 支持浏览器扩展

作者: @monsterxcn

举例: https://rsshub.app/onejav/popular/30

路由: /onejav/:type/:key?

参数:

  • type, 必选 -

    类型

  • key, 可选 -

    关键词

类型

最新 热门 随机 指定演员 指定标签 指定日期
new popular random actress tag day

关键词

空 日期范围 演员名 标签名 日期
7 / 30 / 60 Yua%20Mikami Adult%20Awards YYYYMMDD

示例说明

  • /onejav/new

    仅当类型为 new popular 或 random 时关键词可为 空

  • /onejav/popular/30

    popular random 类型的关键词可填写 7 30 或 60 三个 日期范围 之一

  • /onejav/actress/Yua%20Mikami

    actress 类型的关键词必须填写 演员名 ，可在 此处 (opens new window) 演员单页链接中获取

  • /onejav/tag/Adult%20Awards

    tag 类型的关键词必须填写 标签名 且标签中的 / 必须替换为 %2F ，可在 此处 (opens new window) 标签单页链接中获取

  • /onejav/day/20200730

    day 类型的关键词必须填写 日期 ，按照示例写成形如 20200730 的格式

Tiny Tiny RSS 会给所有 iframe 元素添加 sandbox="allow-scripts" 属性，导致无法加载西瓜视频内嵌视频，如果需要使用内嵌视频请为 Tiny Tiny RSS 安装 remove_iframe_sandbox (opens new window) 插件

# 用户视频投稿

作者: @FlashWingShadow

举例: https://rsshub.app/ixigua/user/video/4234740937

路由: /ixigua/user/video/:uid/:disableEmbed?

参数:

  • uid, 必选 -

    用户 id, 可在用户主页中找到

  • disableEmbed, 可选 -

    默认为开启内嵌视频, 任意值为关闭

# 喜马拉雅

# 专辑（不输出 ShowNote）

支持播客 支持浏览器扩展 支持 RSSBud 仅支持自建

作者: @lengthmin @jjeejj @prnake

举例: https://rsshub.app/ximalaya/album/299146

路由: /ximalaya/album/:id/:all?

参数:

  • id, 必选 -

    专辑 id, 可在对应专辑页面的 URL 中找到

  • all, 可选 -

    是否需要获取全部节目，填入 1、true、all 视为获取所有节目，填入其他则不获取。

目前喜马拉雅的 API 只能一集一集的获取各节目上的 ShowNote，会极大的占用系统资源，所以默认为不获取节目的 ShowNote。下方有一个新的路径可选获取 ShowNote。

注意

专辑 id 是跟在分类拼音后的那个 id, 不要输成某集的 id 了

付费内容需要配置好已购买账户的 token 才能收听，详情见部署页面的配置模块

# 专辑（输出 ShowNote）

支持播客 仅支持自建

作者: @lengthmin @jjeejj @prnake

举例: https://rsshub.app/ximalaya/album/39488639/0/shownote

路由: /ximalaya/album/:id/:all/:shownote?

参数:

  • id, 必选 -

    专辑 id, 可在对应专辑页面的 URL 中找到

  • all, 必选 -

    是否需要获取全部节目，填入 1、true、all 视为获取所有节目，填入其他则不获取。

  • shownote, 可选 -

    是否需要获取节目的 ShowNote，填入 1、true,shownote 视为获取，填入其他则不获取。

# 排行榜

作者: @queensferryme @DIYgod

举例: https://rsshub.app/zimuzu/top/week/movie

路由: /zimuzu/top/:range/:type

参数:

  • range, 必选 -

    时间范围, 可以是 week month year total

  • type, 必选 -

    排行类型, 可以是 fav tv movie

例如，路由 /zimuzu/top/week/movie 应该输出 http://www.rrys2019.com/html/top/week_movie_list.html (opens new window) 的排行榜单
