# Craigslist

# 商品搜索

作者: @lxiange

举例: https://rsshub.app/craigslist/sfbay/sso?query=folding+bike&sort=rel

路由: /craigslist/:location/:type?

参数:

  • location, 必选 -

    位置，即Craigslist的子域，如sfbay

  • type, 可选 -

    搜索类型，如sso

由于 Craigslist 取消了 RSS 订阅搜索功能，因此用 RSSHub 来实现了类似效果。 一个完整原始搜索会像这样： https://sfbay.craigslist.org/search/sso?query=folding+bike&sort=rel (opens new window)

/search/xxx 后跟的 "xxx" 为搜索类型，直接参考原始请求即可。 query string 是实际的搜索内容。

# Mercari

# 商品

作者: @nczitzk

举例: https://rsshub.app/mercari/category/1

路由: /mercari/:type/:id

参数:

  • type, 必选 -

    类型，可选 category 指按类别浏览，brand 指按品牌浏览，search 指搜索关键词浏览

  • id, 必选 -

    id，可在对应分类或品牌页 URL 中找到。若选择 search 作为 type 则此处填写关键词

所有分类参见 分类清单 (opens new window)

所有品牌参见 品牌清单 (opens new window)

# 大麦网

# 票务更新

作者: @hoilc

举例: https://rsshub.app/damai/activity/上海/音乐会/全部/柴可夫斯基

路由: /damai/activity/:city/:category/:subcategory/:keyword?

参数:

  • city, 必选 -

    城市, 如果不需要限制, 请填入全部

  • category, 必选 -

    分类, 如果不需要限制, 请填入全部

  • subcategory, 必选 -

    子分类, 如果不需要限制, 请填入全部

  • keyword, 可选 -

    搜索关键字, 置空为不限制

城市、分类名、子分类名，请参见大麦网搜索页面 (opens new window)
