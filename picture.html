# nHentai

# 分类筛选

支持 BT 反爬严格

作者: @MegrezZhu @hoilc

举例: https://rsshub.app/nhentai/language/chinese

路由: /nhentai/:key/:keyword/:mode?

参数:

  • key, 必选 -

    筛选条件，可选: parody, character, tag, artist, group, language, category

  • keyword, 必选 -

    筛选值

  • mode, 可选 -

    模式，simple为仅封面，detail会包括本子每一页，但对服务器负载大，torrent会包括磁力链接，需要登录，参见部分 RSS 模块配置。默认为simple

# 高级搜索

支持 BT 反爬严格

作者: @MegrezZhu @hoilc

举例: https://rsshub.app/nhentai/search/language%3Ajapanese+-scat+-yaoi+-guro+-%22mosaic+censorship%22

路由: /nhentai/search/:keyword/:mode?

参数:

  • keyword, 必选 -

    用于搜索的关键词。可在原网站搜索后复制 q= 后面的内容，也可直接输入。用法详见官网

  • mode, 可选 -

    模式，simple为仅封面，detail会包括本子每一页，但对服务器负载大，torrent会包括磁力链接，需要登录，参见部分 RSS 模块配置。默认为simple
