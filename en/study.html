# iciba

# Daily English Sentence

作者: @mashirozx

举例: https://rsshub.app/iciba/7/poster

路由: /iciba/:days?/:img_type?

参数:

  • days, 可选 -

    number of items to show（min = 1, max = 7, default = 1）

  • img_type, 可选 -

    image style

:img_type image style
original Original size
medium Medium size
thumbnail Thumbnail size
poster Art poster

# theses

Example: https://rsshub.app/linkresearcher/category=theses&subject=生物

Route: /linkresearcher/theses/:param

Parameters:

  • param, required -

    key=value，eg. subject=生物

:param example definition
category category=thesis one of，theses/information/careers
subject subject=生物 string / undefined
columns columns=健康 string / undefined
columns columns=virus string / undefined
