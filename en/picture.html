# nHentai

# Filter

support BT strict anti-crawler policy

Author: @MegrezZhu @hoilc

Example: https://rsshub.app/nhentai/language/chinese

Route: /nhentai/:key/:keyword/:mode?

Parameters:

  • key, required -

    Filter term, can be: parody, character, tag, artist, group, language, category

  • keyword, required -

    Filter value

  • mode, optional -

    mode, simple to only cover，detail to all content, torrent to include Magnet URI, need login, refer to Route-specific Configurations, default to simple

# Advanced Search

support BT strict anti-crawler policy

Author: @MegrezZhu @hoilc

Example: https://rsshub.app/nhentai/search/language%3Ajapanese+-scat+-yaoi+-guro+-%22mosaic+censorship%22

Route: /nhentai/search/:keyword/:mode?

Parameters:

  • keyword, required -

    Keywords for search. You can copy the content after q= after searching on the original website, or you can enter it directly. See the official website for details

  • mode, optional -

    mode, simple to only cover，detail to all content, torrent to include Magnet URI, need login, refer to Route-specific Configurations, default to simple
