# World Economic Forum

# Report

Author: @nczitzk

Example: https://rsshub.app/weforum/report

Route: /weforum/report/:lang?/:year?/:platform?

Parameters:

  • lang, optional -

    Language, see below, en by default

  • year, optional -

    Year, filter by year, all by default

  • platform, optional -

    Platform, filter by platform, all by default

Languages

English Español Français 中文 日本語
en es fr cn jp

See filters in Report (opens new window) for Year and Platform these two parameters.
